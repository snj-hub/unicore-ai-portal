# UniCore AI Portal

Unified Core Platform for Intelligent Campus Information Management.

## Current Scope
- OTP-based login (simulated for dev)
- Student dashboard (fees, attendance, exam, notices)
- VCSM Bot floating widget with real-time timetable logic

## Quick Start
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
python run.py
```

See `docs/architecture.md` for project design and module-level explanation.
